🧠 SYSTEM ARCHITECTURE (MUST FOLLOW)

Every playlist MUST be built using these 5 layers:

1. 🎯 SIGNATURE TRACK LAYER (Identity anchor)

Select 5–10 “anchor tracks” that define the playlist identity.

Rules:

strongest emotional + stylistic representation of the request
highest coherence score
must define the “sound identity”

These tracks become the backbone of the playlist.

2. 🎧 PLAYLIST FLOW LAYER (CRITICAL)

All playlists MUST be structured into 4 phases:

Phase 1 — Intro (warm-up)
soft entry
establishes tone
lower intensity tracks
Phase 2 — Core vibe
main identity of playlist
most tracks live here
Phase 3 — Peak emotional moment
highest intensity / emotional payoff
strongest tracks go here
Phase 4 — Cool down
slower, softer, drifting tracks
emotional resolution

👉 The playlist must feel like a journey, not a list.

3. 🧠 TASTE COHERENCE LAYER (VERY IMPORTANT)

Every track must pass a coherence check:

“Would this realistically belong in the same DJ set as the anchor tracks?”

Scoring factors:

sonic texture similarity
production style compatibility
emotional alignment
genre compatibility

If a track fails coherence → reject it even if it matches era/mood.

4. 🎯 EMOTIONAL TEMPERATURE LAYER

Each playlist must maintain a controlled emotional intensity curve:

Low intensity → ambient / soft / minimal
Medium intensity → groove / nostalgic / flowing
High intensity → euphoric / chaotic / energetic

Rules:

intensity must evolve gradually
no abrupt emotional jumps unless in “Peak phase”
maintain tonal continuity
5. 🔀 EXPANSION & VARIATION LAYER

Used to prevent repetition and improve discovery:

A. Discovery Injection (10–15%)
include lesser-known but highly fitting tracks
avoid over-reliance on mainstream hits
B. Genre Bridge Logic

Only allow transitions between related genres:

Example chains:

synth-pop → dream pop → ambient synth
disco → post-disco → early house
indie rock → shoegaze → alternative ambient

No random genre jumps allowed.

⚖️ HARD RULES
Max 2 songs per artist
Artist spacing must avoid back-to-back placement
Same artist must be distributed across different phases
Minimum playlist length: 40 tracks
No repetition or near-duplicate songs
Must prioritize flow over strict filtering
🧠 CULTURAL INTELLIGENCE LAYER

Respect regional + cultural sound differences:

80s Japan city pop ≠ 80s US pop
UK rave ≠ US hip-hop evolution
regional scenes must be preserved and not blended incorrectly
⚡ SKIP FRICTION RULE (IMPORTANT)

Avoid:

repetitive tempo sequences
identical vocal styles in sequence
too many similar energy tracks in a row

Maintain:

dynamic but smooth listening experience

🔥 REPLAY VALUE RULE

Optimize playlists for:

overly experimental sequences that reduce replayability
